# 黏液型食蟲植物的祕密 🌿

**探討光照與濕度對黏液型食蟲植物黏液表現與餌料黏附能力之影響 —— 以毛氈苔與捕蟲堇為例**

一個依據第 67 屆科展研究設計整合稿製作的**互動式教學網站**，適合國小中高年級自然科學「植物與環境探究」單元使用。

## ✨ 互動功能

- **八項實驗手冊**：以分頁切換瀏覽每個實驗的自變因／應變因／控制變因與詳細步驟。
- **黏液表現模擬器**：拖動「光照」與「濕度」滑桿，即時看到黏液珠變化、預測黏液分數與餌料黏附成功率（教學示意模型）。
- **互動小測驗**：6 題即時批改，檢驗學習成效。
- **滾動動畫、響應式版面（RWD）**：手機、平板、電腦皆可瀏覽。

## 📚 內容章節

研究動機 → 文獻探討 → 研究目的 → 實驗配置總表 → 八項實驗設計 → 互動模擬 → 資料分析 → 材料與經費 → 預期成果與限制 → 互動測驗 → 參考文獻

## 🚀 使用方式

本網站為單一 `index.html` 自給自足檔案，直接以瀏覽器開啟即可，或透過 GitHub Pages 線上瀏覽。
